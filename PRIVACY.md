# 隐私说明

## 数据范围

个人工作面板用于保存用户主动输入的项目、待办、问题、总结、灵感以及用户主动粘贴的图片。

## 存储位置

- 数据默认保存在当前 Windows 用户的 Tauri 应用数据目录中。
- 应用使用本地 JSON 文件和应用私有图片目录，不使用远程数据库。
- 用户主动创建备份或导出 Markdown 时，文件仅写入用户通过系统对话框选择的位置。

## 网络与账号

- 应用不要求注册或登录。
- 应用不包含云同步、遥测、广告或分析 SDK。
- 核心功能不需要互联网连接。

## 开源仓库边界

- 用户数据目录不位于源码仓库中。
- `.gitignore` 排除本地环境文件、日志、构建产物、安装工具和发布二进制。
- README 截图使用虚构演示数据，不来源于任何真实用户数据文件。

## 用户控制

用户可通过应用的备份功能保留数据副本。卸载或删除数据前，请先创建备份；项目维护者无法访问或恢复用户本机数据。
